Echelon Clicker is a **modern, minimal auto-clicker GUI application** built with Python.  
It allows you to start and stop a click loop with a hotkey, and you can customize the hotkey anytime.  

---

## ‚ö° Features

- **Start / Stop click loop** with a single button
- **Custom hotkey** support (default: F6)
- **Minimal and modern GUI** with dark/magenta theme
- **Hover effects** on buttons
- **Navbar with logo**
- **Fixed 2-second hold time** for clicks (no extra configuration needed)
- **Threaded click loop** for smooth performance
- Works with **Minecraft, Roblox, or any application** that requires click-holding
